40分钟 理论学习 + 20分钟 实践练习

建议时间：晚上7-8点（固定时间更容易坚持）

2-3周
26个字母 编程关键词 常用函数名 错误信息理解 代码阅读

  • 📱 手机APP：百词斩、扇贝单词（学编程单词）

  • 📖 在线工具：有道翻译、百度翻译

4-6周

6-8周

阶段6：求职准备
2-3周
简历制作 面试准备 技术总结 作品集整理 面试技巧

📚 求职资源

  • 💼 招聘网站：BOSS直聘、拉勾网、智联招聘
  • 📝 简历模板：超级简历、五百丁
  • 🎯 面试题库：牛客网、力扣
  • 📱 技术社区：CSDN、博客园

🚨 英文零基础特别提醒

  • 不要害怕英文：编程用的英文很简单，就那100-200个单词
  • 准备翻译工具：手机装有道翻译，电脑开着百度翻译
  • 制作单词本：把遇到的编程词汇记下来，反复看
  • 从中文教程开始：先看中文视频，再看英文文档
  • 错误信息不要怕：复制错误信息到百度搜索中文解答

📝 必背编程单词（前50个）

基础词汇：
print 打印、input 输入、if 如果、else 否则、for 循环、while 当、def 定义、return 返回、import 导入、from 从

数据类型：
int 整数、str 字符串、list 列表、dict 字典、len 长度、range 范围、True 真、False 假、None 空

爬虫词汇：
request 请求、response 响应、url 网址、data 数据、text 文本、html 网页、get 获取、post 发送

💡 学习建议

  • 坚持每天学习：1小时很短，但坚持7个月就能看到效果
  • 多动手实践：看视频的同时一定要跟着敲代码
  • 遇到问题不要慌：百度、CSDN、知乎都是好帮手
  • 建立学习笔记：用有道云笔记或者语雀记录重点
  • 加入学习群：找一些Python学习QQ群或微信群
